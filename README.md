# お薬管理カレンダー（Medication Calendar）

## 概要
薬の名前と、飲むタイミング（朝/昼/晩、食前/食後）を登録すると、
カレンダーに服用予定が表示されるWebアプリを作成します。

## 主な機能（予定）
- 服用予定の登録（日付、薬名、朝/昼/晩、食前/食後、メモ）
- 日別一覧表示（まずはここまでをMVPとする）
- 月間カレンダー表示（次の段階で実装）
- 編集・削除（拡張）
